logcheck-filter

Filter logs using logcheck rules

**Usage:** `logcheck-filter [OPTIONS] --rules `

###### Subcommands:

  • file — Read from a file
  • stdin — Read from standard input
  • journald — Read from systemd journal

###### Options:

  • `--rules ` — Path to logcheck rules directory
  • `--format ` — Output format

Default value: text

Possible values:

  • text:

Human-readable text format

  • json:

JSON format

  • `--show ` — What entries to show

Default value: all

Possible values:

  • all:

Show all log entries

  • violations:

Show only violations (cracking/violations)

  • unmatched:

Show only unmatched entries

  • --stats — Show processing statistics
  • --color — Enable colored output
  • `--output-file ` — Write filtered logs to file
